# Datalytics

### Download packages:

```
bower install
npm install
```

### Build:

```
gulp
```

### Run:

```
npm start
```
if there are any errors run

```
echo fs.inotify.max_user_watches=524288 | sudo tee -a /etc/sysctl.conf && sudo sysctl -p
```

and try again

### Run homepage

http://localhost:3010
